# Поздравительный лендинг "С Новым 2026"

Одностраничный лендинг с 5 экранами, мини-игрой и интерактивной конфетой для поздравления клиентов.

## Технологии

- **Next.js 15** (App Router)
- **React 18** + **TypeScript**
- **Framer Motion** (анимации)
- **Tailwind CSS** (стилизация)

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен-версии
npm start
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Особенности

### Адаптивность
- Мобильная версия (360px+)
- Планшет (768px+)
- Десктоп (1280px+)

### Доступность (A11y)
- Поддержка `prefers-reduced-motion` (упрощённые анимации)
- Навигация с клавиатуры
- ARIA-метки для интерактивных элементов
- Контрастность текста (WCAG AA)

### Аналитика
События отслеживаются в `lib/analytics.ts`. Для подключения GA4/Я.Метрики:
1. Раскомментируйте код в `analytics.ts`
2. Добавьте скрипты аналитики в `app/layout.tsx`

### Контент
Все тексты хранятся в `content/landing.ru.json` и могут редактироваться без изменения кода.

## Ассеты от дизайнера

Необходимо добавить в `public/`:
- Фоны для экранов (турбулентность, спокойный фон)
- Фото команды (15-30 шт) в формате WebP/AVIF
- Спрайты молотка и эффектов трещин
- Изображения конфеты (состояния: closed, opening, open)
- OG-изображение (1200×630)
